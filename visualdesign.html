Visual design / digital typography

This project is a part of the master course at Scuola Politecnica di Design. The Trumpet is an online e-commerce website that had to be redesigned and re-branded.

The idea

The goal is to design a new logo using only typography for it and give a new look and feel for the whole website.

Home page

Home page is designed with colors and fonts. This page is divided into several sections where each of them is for different aims.

Editorial

Editorial page is designed with pastel color pallet. This idea gives the audience an authentic look of the website and also luxury and elegant feeling. All the details including images and products are places with the idea to underline the luxury of the brand.

Hot this week

Hot this week is created for showing what is popular this week. The chosen color is red, so all the products include red.

Catswalk

Catswalk page is designed with red color which highlights the latest news in the fashion world.

Icons page

Icon page has a simple but elegant design. This page is divided into sections where each section is highlighted with the title. All the pictures of famous people are in black and white and the products are in color option which gives the special luxury feeling.
