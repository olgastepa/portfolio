Web design for a design studio

The goal is to create a website for a digital design studio based in Saint-Petersburg, Russia.

The idea

The idea of the project is to make a website with a simple interface mixed with an elegant look and creative feel. Produce a website with a user-friendly interface by adding edited in the same style images and colors of the brand.

Home page

The home page has a very simple interface but it looks creative and elegant. It has an introduction about the company, social media links, and navigation.

Services page

List of services

The list of services is designed in a precise way to highlight what the company works with.

About us

The page gives the audience an idea about the studio. This page, as well as all other pages, is created with images and titles on it.

Contact page

The contact page has the same design as the home page. There are two main colors of the brand and image. On the image, there are contacts with links.
