App design for ATM Milano

This app design project is a part of the master course at Scuola Politecnica di Design. The main idea of this app design is to redesign the current mobile app. This project includes the research stage, testing, UX and UI design. ATM Milano is a public transport company in Milan, Italy. This app works for all the public transport in Milan.

The idea

The goal of this mobile app is to redesign the current mobile app inventing new functions and make the interface user-friendly. The new mobile app has been designed, which gives users a new travel experience. Our mobile app allows users to buy tickets with different payment methods, see the route, check the availability of a ticket (it is still valid or not), repurchase previous tickets, buy travel cards (cards more unlimited use for a student, elderly people, etc). Also, the app helps with navigation which means it will give you the best route on how to get from point A to point B and will tell you which type of ticket you’ll need for it.

Logo & home page

There is a logo of the client and the menu page

Menu

The menu page is designed accordingly to the main colors and new branding idea of the app. This page has a user-friendly interface which makes the app easy to use people with a wide range of age. It has six major sections and on the top right corner, there is an icon for saved tickets.

Ticket

The ticket page has a QR code on the online ticket. On this page, users have a choice. There is an option to scan the ticket when you start using public transport. Also, you can sign up to save this ticket for later.

Travel card

A travel card is a card for young, retired people, etc. The online card is shown with client details. In the app, the travel card is available online. On this page, users can make the prepayment for the next month and check how many days are left.

My trips

My trips are created to make users experience much easier. There is an option to re-purchase expired tickets which were used before. Also, users can see which tickets are available right now.

Checkout

The checkout page is a part of the payment process. On this page, there are four options for how it is possible to buy a ticket. There are SMS, PayPal, credit card and apple pay.
